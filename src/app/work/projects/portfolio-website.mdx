## Modern Portfolio Website

A sleek, responsive portfolio website showcasing my work, skills, and experiences. Built with Next.js and Once UI design system.

### Key Features

- **Responsive Design**: Seamless experience across all devices
- **Dark/Light Mode**: User preference-based theme switching
- **Interactive UI**: Smooth animations and transitions
- **SEO Optimized**: Structured data and meta tags for better search visibility
- **Performance Focused**: Optimized for fast loading and interaction
- **Accessibility**: WCAG compliant for all users

### Technology Stack

- **Frontend**: Next.js 14, React, TypeScript
- **Styling**: Tailwind CSS, Once UI System
- **Content Management**: MDX for project content
- **Deployment**: Vercel
- **Analytics**: Plausible Analytics (privacy-focused)

### Development Process

I built this portfolio to showcase my skills and projects in a modern, clean interface. The design emphasizes readability and user experience while maintaining fast performance scores. 

Special attention was paid to accessibility features and responsive design to ensure all visitors have a great experience regardless of their device or abilities.

### Results

The portfolio has received positive feedback from peers and potential employers, leading to several interview opportunities. It achieves a 98/100 Lighthouse score for performance and accessibility.
